# 📅 Day 4 - NumPy & Pandas for AI/ML Internship

## 📖 Overview

On Day 4 of my AI & ML Learning Internship, I learned the fundamentals of **NumPy** and **Pandas**, two of the most essential Python libraries used in Data Science, Artificial Intelligence, and Machine Learning.

I completed multiple hands-on practice programs and built a **Student Performance Analysis Project** using a real-world CSV dataset. Through this project, I learned how to load, explore, analyze, and process data efficiently using Pandas while performing numerical computations with NumPy.

---

# 📄 Dataset

The dataset used in this project contains **50 student records** stored in a CSV file.

It includes student information such as:

- Rank
- Roll Number
- Student Name
- Mathematics Marks
- Physics Marks
- English Marks
- Computer Science Marks
- Total Marks

The dataset was loaded using **Pandas**, explored using different functions, filtered based on conditions, analyzed statistically, and finally saved as a processed CSV file.
