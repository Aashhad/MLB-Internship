# Student Record Management System (Day 2)

## 📌 Project Description

This project is a simple **Student Record Management System** developed using Python. It is a menu-driven console application that demonstrates the use of Python fundamentals such as lists, dictionaries, functions, loops, and conditional statements. The application stores student records in memory and allows users to perform basic CRUD operations.

---

## 🚀 Features

- Add Student
- View All Students
- Search Student by Roll Number
- Update Student Information
- Delete Student
- Prevent Duplicate Roll Numbers
- Display Student Records in a Structured Format

---

## 🛠️ Technologies Used

- Python 3.12.6
- Visual Studio Code

---

## 📂 Student Information

Each student record contains:

- Name
- Roll Number
- Age
- Course

{

1. Open the project folder in **Visual Studio Code**.

## ⚡ Challenges Faced

During this project, I faced the following challenges:

- Understanding how to store multiple student records.
- Preventing duplicate roll numbers.
- Searching students by roll number.
- Updating existing records.
- Deleting records correctly.
- Organizing the code using functions.

---

## ✅ Solutions Implemented

To overcome these challenges, I:

- Used a **list** to store multiple student records.
- Used a **dictionary** to store each student's information.
- Created a helper function (`find_student()`) to search students by roll number.
- Added duplicate roll number validation.
- Used functions to keep the code modular and reusable.
- Built a menu-driven interface using a `while` loop.
