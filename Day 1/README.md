# 💻 Mini Project

## Student Grading System

Developed a Python program that:

- Takes the student's name as input.
- Takes the class name as input.
- Accepts the number of subjects.
- Takes subject names and marks.
- Calculates the average marks.
- Assigns grades based on the average.

---

# 📊 Grade Criteria

| Average Marks | Grade |
|--------------:|:-----:|
| 90 – 100 | A |
| 75 – 89 | B |
| 50 – 74 | C |
| Below 50 | F |
